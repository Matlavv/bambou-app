# Bambou

## Install project

To install the project, follow these steps:

1. Navigate to the project directory:

```bash
cd bambou
```

2. Install dependencies:

```bash
npm install
```

or

```bash
yarn install
```

## Dependencies

The project has the following dependencies:

- Firebase
- NativeWind
- React Navigation
